# 概览页面完善计划

## 一、当前页面分析

### 1.1 现有功能模块

| 模块 | 功能描述 | 状态 |
|------|----------|------|
| 本周行动建议 | 根据数据智能推荐优先处理事项 | ✅ 已上线 |
| 核心指标卡片 | 展示家庭总任务、待审核、待核销、可用总积分 | ✅ 已上线 |
| 完成质量/按时率/处理压力 | 三个核心KPI指标展示 | ✅ 已上线 |
| 任务状态分布 | 进行中/待审核/已完成/已驳回的进度条可视化 | ✅ 已上线 |
| 任务类型分布 | 日常/进阶/挑战任务的分布展示 | ✅ 已上线 |
| 近7天完成趋势 | 柱状图展示最近7天任务完成情况 | ✅ 已上线 |
| 孩子表现面板 | 每个孩子的任务统计和快捷入口 | ✅ 已上线 |
| 关键结果看板 | 按时完成/逾期完成/待审核/已完成的数量展示 | ✅ 已上线 |

### 1.2 当前存在的问题

#### 问题1：数据维度单一
- 只有任务完成数量统计，缺乏积分流转数据
- 缺少孩子之间的对比分析
- 没有时间维度的趋势对比（如环比、同比）

#### 问题2：缺乏激励体系展示
- 没有展示成就系统数据
- 缺少连续打卡/习惯养成数据
- 没有展示排名/竞争元素

#### 问题3：交互深度不足
- 图表都是静态展示，无法下钻查看详情
- 缺少数据筛选能力（按时间范围、按孩子筛选）
- 没有数据导出功能

#### 问题4：实时性不够
- 数据需要手动刷新页面才更新
- 缺少实时通知/提醒机制

#### 问题5：视觉层次可优化
- 信息密度较高，重点不够突出
- 缺少数据为空/异常状态的友好提示
- 响应式布局在移动端体验待优化

## 二、完善计划

### 阶段一：数据丰富度提升（优先级：高）

#### 2.1 新增积分流转看板
**功能描述**：展示积分的获取和消耗情况
- 本周发放积分总额
- 本周兑换消耗积分
- 积分余额变化趋势
- 积分获取/消耗TOP任务

**实现方案**：
```typescript
interface PointsFlowData {
  issuedThisWeek: number;
  redeemedThisWeek: number;
  balanceTrend: { date: string; balance: number }[];
  topTasksByPoints: { taskName: string; points: number; count: number }[];
}
```

**预计工时**：2天

#### 2.2 新增习惯养成追踪
**功能描述**：展示孩子的习惯任务坚持情况

- 习惯完成率
- 习惯中断记录
- 习惯养成趋势

**实现方案**：
- 复用现有的 habits API
- 新增习惯统计数据聚合接口

**预计工时**：2天

#### 2.3 新增成就展示模块
**功能描述**：展示家庭成就系统数据

## 三、技术实现方案

### 3.1 新增API接口

```typescript
// GET /api/stats/overview
interface OverviewStatsParams {
  startDate?: string;
  endDate?: string;
  childId?: string;
}

interface OverviewStatsResponse {
  // 任务统计
  tasks: {
    total: number;
    pending: number;
    submitted: number;
    approved: number;
    rejected: number;
    byType: { daily: number; advanced: number; challenge: number };
    onTimeRate: number;
    completionRate: number;
    trend: { date: string; count: number }[];
  };
  // 积分统计
  points: {
    issued: number;
    redeemed: number;
    balance: number;
    trend: { date: string; issued: number; redeemed: number }[];
  };
  // 习惯统计
  habits: {
    streakDays: number;
    completionRate: number;
    totalHabits: number;
  };
  // 成就统计
  achievements: {
    unlocked: number;
    total: number;
    recent: Achievement[];
  };
  // 孩子对比数据
  childrenComparison: {
    childId: string;
    name: string;
    approvedTasks: number;
    pointsEarned: number;
    onTimeRate: number;
    achievements: number;
  }[];
}
```
